📦 cPanel Dosyalarını İndir

Web sitenizi cPanel'e yüklemek için gerekli tüm dosyalar hazır!

📁 Paket İçeriği:

  • index.html - Ana sayfa
  • style.css - Stil dosyası
  • script.js - JavaScript dosyası
  • .htaccess - Sunucu ayarları
⬇️ ZIP Dosyasını İndir 📖 Yükleme Rehberini İndir

ZIP dosyasını indirdikten sonra cPanel File Manager'da public_html klasörüne yükleyin.
